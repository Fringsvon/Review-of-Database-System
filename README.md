# 同济大学数据库复习资料

## 课程说明

同济大学电信学院计算机系《数据库系统原理》课程，课号100396，计算机科学与技术、信息安全专业必修（大三上）。

本项目提供了本人搜集整理的复习资料和作业答案，供学弟学妹参考。

考试卷子是英文，闭卷，不可以带字典，但是可以用中文作答。

## 内容说明

本人18降转19，但是这门课还是跟18一起上的，Zhang班级。

参考试卷下载自百度文库，还有一部分参考题目是Xiang当年上课展示的。

18级开始，Li取代Xiang负责课程，因此这些以前的卷子题量较少，题目难度也比近两年低很多。

但是Xiang时期几份卷子里的英文概念不定项选择题，会有较大重合率。

期末复习划重点的PPT，一份是Li提供的，另一份来自百度文库，可能是软件学院的。

另外还有一部分本人提供的手写笔记，主要总结了以下知识点的解题步骤，都是大题必考项目。

+ 范式分析 (函数闭包计算，正则覆盖，主键计算，无损连接判断，BCNF/3NF/4NF判定及分解算法)
+ B+树（写查找过程，画出插入删除过程）
+ 系统崩溃后根据日志两阶段恢复算法（根据日志，写恢复过程每一步撤销什么，重做什么）
